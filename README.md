Login System - BSD 214

This project is a simple Login System implemented using Java.

Features:
- User authentication using username and password
- Maximum of 3 login attempts
- Simple test file for validation

Files:
- LoginSystem.java : Main login program
- LoginTest.java   : Test cases for login method

How to Run:
1. Compile all Java files
2. Run LoginSystem.java
3. Run LoginTest.java for testing
